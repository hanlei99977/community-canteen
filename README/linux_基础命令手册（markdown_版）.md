# Linux 基础命令手册（Markdown 版）

> 适用环境：Ubuntu / Debian 系 Linux  
> 适用人群：Linux 初学者 / C++ & Web 项目开发 / 课程设计

---

## 一、目录与文件操作

| 命令 | 作用 |
|------|------|
| `pwd` | 显示当前所在目录 |
| `ls` | 列出当前目录内容 |
| `ls -l` | 详细列表显示文件信息 |
| `ls -a` | 显示所有文件（含隐藏文件） |
| `ls -lh` | 以易读方式显示文件大小 |
| `cd 目录` | 切换到指定目录 |
| `cd ~` | 切换到用户家目录 |
| `cd ..` | 返回上一级目录 |
| `cd -` | 返回上一次所在目录 |
| `mkdir 目录名` | 创建目录 |
| `mkdir -p a/b/c` | 创建多级目录 |
| `touch 文件名` | 创建空文件 |

---

## 二、文件操作

| 命令 | 作用 |
|------|------|
| `cp 源 目标` | 复制文件 |
| `cp -r 源 目标` | 递归复制目录 |
| `mv 源 目标` | 移动或重命名文件/目录 |
| `mv -i 源 目标` | 移动前确认（防止覆盖） |
| `rm 文件` | 删除文件 |
| `rm -r 目录` | 删除目录及其内容 |
| `rm -ri 目录` | 删除前逐个确认 |
| `file 文件名` | 查看文件类型 |

---

## 三、查看文件内容

| 命令 | 作用 |
|------|------|
| `cat 文件` | 查看文件全部内容 |
| `less 文件` | 分页查看文件内容（推荐） |
| `more 文件` | 分页查看文件 |
| `head 文件` | 查看文件前 10 行 |
| `head -n 20 文件` | 查看文件前 20 行 |
| `tail 文件` | 查看文件后 10 行 |
| `tail -n 50 文件` | 查看文件后 50 行 |
| `tail -f 文件` | 实时查看文件变化 |

---

## 四、查找与搜索

| 命令 | 作用 |
|------|------|
| `find 路径 -name 文件名` | 按名称查找文件 |
| `find . -name "*.cpp"` | 查找当前目录下的 cpp 文件 |
| `grep 关键词 文件` | 在文件中搜索内容 |
| `grep -R 关键词 目录` | 递归搜索目录内容 |
| `which 命令` | 查看命令所在路径 |
| `whereis 命令` | 查找命令相关文件 |

---

## 五、权限与用户

| 命令 | 作用 |
|------|------|
| `chmod 权限 文件` | 修改文件权限 |
| `chmod +x 文件` | 添加执行权限 |
| `chmod 755 文件` | 设置常用执行权限 |
| `chown 用户 文件` | 修改文件所有者 |
| `chown 用户:组 文件` | 修改文件所有者和组 |
| `sudo 命令` | 以管理员权限执行命令 |

---

## 六、用户与系统信息

| 命令 | 作用 |
|------|------|
| `whoami` | 显示当前用户 |
| `id` | 查看用户与用户组信息 |
| `uname -a` | 查看系统信息 |
| `df -h` | 查看磁盘使用情况 |
| `free -h` | 查看内存使用情况 |

---

## 七、软件管理（Ubuntu）

| 命令 | 作用 |
|------|------|
| `sudo apt update` | 更新软件源 |
| `sudo apt upgrade` | 更新已安装软件 |
| `sudo apt install 软件名` | 安装软件 |
| `sudo apt remove 软件名` | 卸载软件 |
| `apt search 关键词` | 搜索软件 |

---

## 八、进程与服务管理

| 命令 | 作用 |
|------|------|
| `ps aux` | 查看所有进程 |
| `top` | 实时查看进程和资源 |
| `kill PID` | 结束指定进程 |
| `kill -9 PID` | 强制结束进程 |
| `systemctl status 服务` | 查看服务状态 |
| `systemctl start 服务` | 启动服务 |
| `systemctl restart 服务` | 重启服务 |
| `systemctl stop 服务` | 停止服务 |

---

## 九、网络相关

| 命令 | 作用 |
|------|------|
| `ip addr` | 查看 IP 地址 |
| `ping 地址` | 测试网络连通性 |
| `netstat -tuln` | 查看端口占用 |
| `lsof -i:端口号` | 查看端口对应进程 |
| `ssh 用户@IP` | 远程登录服务器 |

---

## 十、压缩与解压

| 命令 | 作用 |
|------|------|
| `tar -cvf a.tar 文件` | 打包文件 |
| `tar -xvf a.tar` | 解包 tar 文件 |
| `tar -czvf a.tar.gz 文件` | 打包并压缩 |
| `tar -xzvf a.tar.gz` | 解压 tar.gz |
| `zip a.zip 文件` | 压缩为 zip |
| `unzip a.zip` | 解压 zip |

---

## 十一、终端快捷键

| 快捷键 | 作用 |
|------|------|
| `Tab` | 自动补全命令/路径 |
| `↑ / ↓` | 查看历史命令 |
| `Ctrl + C` | 终止当前命令 |
| `Ctrl + L` | 清屏 |
| `Ctrl + D` | 退出终端 |
